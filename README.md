# ML Dataset Analyzer

Web app that automatically analyzes datasets for Machine Learning:
- Target inference
- Classification vs regression detection
- Missing values analysis
- Constant / ID / leakage feature detection
- Class imbalance detection (binary & multiclass)
- Dataset viability score
- Automatic visualizations

**Tech stack**: Python, Flask, Pandas, Matplotlib, HTML/CSS/JS

Upload a CSV and get an instant ML diagnostic.
